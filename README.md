## ğŸ“Š Key KPIs (As Shown in Dashboard)

The dashboard displays the following KPIs **with the same naming convention used in Power BI**:

- **Sum of Amount (Total Sales Revenue)**  
  Represents the total revenue generated from all ecommerce transactions.

- **Sum of Quantity (Total Units Sold)**  
  Indicates the total number of products sold.

- **Sum of Profit (Overall Profit)**  
  Shows the cumulative profit across all orders.

- **Sum of AOV (Order Value Metric)**  
  Represents the aggregated order value metric available in the dataset (not a calculated average).
